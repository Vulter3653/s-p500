# 2024

2024년 S&P 500 구성기업과 해당 기업의 10-K 자료를 관리하는 폴더다.

- 구성기업 확정 기준일: `2025-01-01`
- 기업 중복 제거 기준: SEC CIK
- 세부 표본 규칙: `../docs/sample-definition.md`
- 기업 목록: `sp500_companies.csv` - 500개 기업
- 종목 감사 목록: `sp500_securities.csv` - 503개 종목
- 생성 방법: `../docs/constituent-data-method.md`

데이터 수집 구조가 확정되기 전까지 원본 10-K나 임시 결과를 이 파일과 혼합하지 않는다.
